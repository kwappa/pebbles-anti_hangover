= pebbles-anti_hangover

* https://github.com/kwappa/pebbles-anti_hangover

== DESCRIPTION:

* 体重と睡眠時間、種類から「二日酔いせずに飲める酒量」を計算します。
 * http://www.nikkeibp.co.jp/archives/347/347655.html

== FEATURES/PROBLEMS:

* (あとでかく)

== SYNOPSIS:

  (あとでかく)

== REQUIREMENTS:

* pebbles

== INSTALL:

* gem install pebbles-anti_hangover
